## Installation

NXFramework is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NXFramework'
or
pod "NXFramework" ,:git => 'https://github.com/jsonwang/NXFramework.git'
```
